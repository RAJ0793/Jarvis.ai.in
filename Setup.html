JARVIS Setup

Your AI Assistant Installation Guide

Get the Latest Release Required

Download the latest version of Python from the official website. Python 3.10.0 or later is required for optimal JARVIS performance.

Visit Python.org

Installation Instructions Step by step

  1. Install Python: Ensure Python 3.10.0 or later is installed. Important: Check "Add Python to PATH" during installation.
  2. Get JARVIS: Download or clone the JARVIS project from GitHub Repository
  3. Open Terminal: Navigate to the extracted project folder using command prompt or terminal.
  4. Install Dependencies: Run pip install -r requirements.txt
  5. Launch JARVIS: Start your AI assistant with python main.py

Troubleshooting
